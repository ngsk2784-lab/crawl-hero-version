Crawl Hero 개인정보처리방침

최종 수정일: 2026-05-27

1. 수집하는 개인정보 항목

항목수집 시점이용 목적
Google 계정 이메일로그인 시계정 식별
Google 사용자 ID (UID)로그인 시계정 식별, 클라우드 세이브 키

2. 수집 방법

  • Google OAuth 2.0을 통한 로그인 (Supabase Authentication 사용)
  • 게임 플레이 중 자동으로 수집되는 진행 데이터

3. 보관 및 처리

  • 저장 위치: Supabase (한국 리전, AWS 인프라)
  • 전송: 모든 트래픽 HTTPS/TLS 암호화
  • 접근 제어: Row Level Security로 본인 데이터만 접근 가능
  • 보관 기간: 계정 삭제 시까지. 계정 삭제 후 즉시 영구 삭제

4. 제3자 제공

개인정보를 제3자에게 제공하지 않습니다. 단, 다음 서비스 제공자가 데이터 처리를 위해 접근합니다:

  • Google LLC — OAuth 인증
  • Supabase Inc. — 데이터베이스 호스팅

5. 이용자 권리

이용자는 다음 권리를 행사할 수 있습니다:

  • 열람: 게임 내 통계 화면에서 본인 데이터 확인
  • 정정: 게임 내에서 직접 수정 (장비, 캐릭터 등)
  • 삭제: 게임 내 설정 → 데이터 초기화로 즉시 영구 삭제 (로컬 + 클라우드)
  • 철회: 로그아웃 후 앱 삭제

6. 쿠키 / 트래커

본 게임은 광고, 분석 트래커, 쿠키를 사용하지 않습니다.

7. 아동 개인정보

본 게임은 만 13세 미만 아동을 대상으로 하지 않습니다. 13세 미만이 등록한 사실을 알게 될 경우 즉시 해당 계정 데이터를 삭제합니다.

8. 보안

  • 전송 구간: TLS 1.2+
  • 저장 구간: Supabase Postgres 암호화
  • 인증: Google OAuth + JWT 토큰
  • 접근 제어: Row Level Security 정책

9. 정책 변경

본 방침이 변경될 경우 앱 내 알림 또는 본 페이지를 통해 공지합니다.

Crawl Hero Privacy Policy

Last updated: 2026-05-27

1. Information We Collect

ItemWhenPurpose
Google account emailOn sign-inAccount identification
Google user ID (UID)On sign-inAccount ID, cloud save key
Game progress dataWhile playingCloud save, leaderboard
Display nameOn sign-inLeaderboard display

2. How We Collect

  • Google OAuth 2.0 sign-in (via Supabase Authentication)
  • Automatic collection of gameplay progress while playing

3. Storage & Processing

  • Location: Supabase (Seoul region, AWS infrastructure)
  • Transit: All traffic encrypted via HTTPS/TLS
  • Access control: Row Level Security — only the account owner can access their data
  • Retention: Until account deletion. Permanently erased immediately upon deletion.

4. Third-Party Sharing

We do not share personal data with third parties. Data is only processed by the following service providers:

  • Google LLC — OAuth authentication
  • Supabase Inc. — database hosting

5. User Rights

  • Access: View your own data in the in-game Stats screen
  • Correction: Modify directly in-game (gear, character, etc.)
  • Deletion: Settings → Reset Data permanently deletes local + cloud immediately
  • Withdrawal: Sign out and uninstall the app

6. Cookies / Trackers

This game does not use ads, analytics trackers, or cookies.

7. Children's Privacy

This game is not directed at children under 13. If we become aware that a child under 13 has registered, we will immediately delete that account's data.

8. Security

  • Transit: TLS 1.2+
  • At rest: Supabase Postgres encryption
  • Authentication: Google OAuth + JWT tokens
  • Access control: Row Level Security policies

9. Policy Changes

If this policy changes, we will notify users in-app or via this page.
